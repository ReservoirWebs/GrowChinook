Welcome to GrowChinook!

This site presents a linked visual foraging and bioenergetics model aimed at exploring growth potential of reservoir rearing juvenile Chinook Salmon. Details on the underlying models, parameter values, data inputs can be found here here.
Once you have read the About page, check out the Instructions for a walkthrough.
Once you are ready, you can:
Run Model with user inputs.
Run Model With Sensitivity using a range for one input variable to explore sensitivity.
Run Advanced Sensitivity using a range for one input variable over observed background data for March through April in 2015. Note: This option is computationally time consuming, please be patient.
Run Multiple Months for preset conditions and scenarios.
