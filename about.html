This linked model uses Python 2 and the NumPy and SciPy packages (www.python.org) with the formulas from the Wisconsin Bioenergetics model (Hanson et al. 1997) and the lowest foraging predicted from 1) a visual encounter (Gregory and Northcote 1993) or 2) functional response (Haskell et al. 2017) foraging model for juvenile Chinook Salmon feeding on plankton. The models are linked using foraging to limit the proportion (P) of maximum consumption (Cmax), when foraging is lower (Figure 1).

From the Wisconsin model, where ƒ(T) is a temperature dependence function:
Growth = Consumption - (Respiration + Waste)
Consumption = Cmax * P * ƒ(T)

To link the models:
P = minimum of (encounter rate / Cmax) or 1

Refer to Table 1 for a full list of parameter values used. The model has been modified to accommodate vertical profile inputs, day and night light levels, and diel vertical migration by Chinook Salmon juveniles. It has also been structured to allow for user determined restrictions in depth or temperatures.

By default, the model seeks to achieve maximum growth for a single juvenile Chinook Salmon given the input conditions. The optimization routine compares day and night depth combinations and determines the maximum growth which can be achieved before compounding daily growth.

Input data were collected from 2013-2016 from 3 reservoirs in the upper Willamette Basin, Oregon (Fall Creek Reservoir, 43°56'33"N 122°45'25"W, Hills Creek Reservoir, 43°40'16"N 122°25'33"W, and Lookout Point Reservoir, 43°53'48"N 122°43'34"W; Figure 2). Collections included light, temperature, zooplankton size, composition, and distribution. Light was collected using a LiCor Li-192 sensor. Extinction coefficients were calculated using the exponential decay curve fits of light for each reservoir and month. Zooplankton vertical distribution was collected at 2 m depths using a 5L transparent Van Dorn sampler where samples were enumerated and identified to species. Zooplankton size and abundance was calculated using a 0.5 m diameter vertically integrated 64 ï¿½m tow net, where samples were identified to species and length measurements were collected for a subset of each species.
